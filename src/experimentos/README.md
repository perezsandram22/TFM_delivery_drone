# 🧪 Rama de Experimentos Visuales

## Propósito
Esta rama está dedicada a probar nuevas ideas, visualizaciones y funcionalidades experimentales sin afectar el código estable de las otras ramas.

## Experimentos planeados
- [ ] Visualización 3D de trayectorias
- [ ] Interfaz gráfica simple para monitoreo
- [ ] Algoritmos de control alternativos
- [ ] Simulación con diferentes modelos de dron
- [ ] Integración de cámara virtual

## Notas
- Todo el código aquí es experimental
- Puede no funcionar o estar incompleto
- Las ideas exitosas se migrarán a ramas específicas
